# Flight Reservation System 

## Project Description

This is a console-based flight reservation system developed using Python.
The system allows passengers to register, log in, book flights, and check their booking status. Cashiers can approve or cancel tickets.

The project demonstrates the use of Object-Oriented Programming (OOP), dictionaries, loops, and conditional statements in Python.

## Features

* Passenger registration (Sign Up)
* Passenger login authentication (Sign In)
* View available flights and seat details
* Book flight tickets (with pending status)
* Cancel booked tickets
* Check booking status
* Cashier approval for ticket confirmation
* Cashier can cancel passenger bookings
* Automatic seat update after booking/cancellation
* Prevents booking when seats are unavailable
  
## Technologies Used

* Python 
* Object-Oriented Programming (OOP)
* Dictionaries and Lists
* Loops and Conditional Statements
* Console Input/Output
 
## System Modules

### Passenger Module
